# Dotfiles

Targeted for OSX. Managed with [rcm][1].

## Contains
[prezto][3]
karabiner
osx keyboard shortcuts

## Install

```
git clone https://github.com/jdsutherland/dotfiles.git ~/.dotfiles
~/.dotfiles/install-script.sh
```
